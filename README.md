\# Animal Company Native Menu (ImGui)



\## Setup



1\. Extract the required ImGui files into `jni/imgui/`

2\. Push to GitHub

3\. Go to Actions tab and download the built `.so`



\## Required ImGui Files



Copy these from `imgui-1.92.8` into `jni/imgui/`:



\*\*Core files:\*\*

\- imgui.cpp, imgui.h, imgui\_draw.cpp, imgui\_widgets.cpp, imgui\_tables.cpp

\- imgui\_internal.h, imconfig.h

\- imstb\_rectpack.h, imstb\_textedit.h, imstb\_truetype.h



\*\*Backends:\*\*

\- backends/imgui\_impl\_android.cpp + .h

\- backends/imgui\_impl\_opengl3.cpp + .h
